About

The past decade has witnessed an unprecedented increase in the intelligent processing of text and audio-visual signals. Although these efforts have made significant advances in improving user experience, a more immersive multimedia system still requires the integration of emerging multi-sensorial media (e.g., text, audio-visual, haptics, olfaction, human motion, and other new media types) with cross-modal interactions. These media modalities enhance the diversity of user perceptions and shift the user's role from observer to participant in multimedia systems. Effectively processing these new media signals and seamlessly incorporating them into state-of-the-art multimedia systems will be critical. This necessitates advancements across the entire multi-sensorial system, as well as considerations of human factors and ergonomics to support implementation. The 1st International Workshop on Multi-Sensorial Media and Applications (MSM), which aims to attract contributions related to multi-sensorial media and its applications, will be colocated with ACM Multimedia 2025 in Dublin, Ireland.

Call for Papers

This workshop welcomes submissions include but are not limited to:

  • Design/Architecture/Implementation of Multi-Sensorial Media System/Environment
  • Human Factors, Ergonomics and Psychophysical Modeling in Multi-sensorial Media
  • Coding and Delivery of Multi-Sensorial Media
  • Analysis and Interpretation of Multi-Sensorial Media
  • Multi-Modal Interaction and Cross-modal Analysis
  • Applications of Generative AI in Multi-Sensorial Media
  • Network Management, Resource Provisioning, and Privacy Issues
  • Standardization of Multi-Sensorial Media Processing
  • Other Emerging Applications of Multi-Sensorial Media
